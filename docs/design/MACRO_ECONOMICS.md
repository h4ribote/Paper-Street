# Macro Economic Indicators (マクロ経済指標)

本ドキュメントでは、Paper Streetにおけるマクロ経済指標の定義、算出アルゴリズム、および市場への影響メカニズムについて記述します。
これらの指標は、プレイヤーの投資判断における最も重要なファンダメンタルズ情報となります。

## 1. 指標一覧 (Indicators Overview)

| 指標名 | 略称 | 更新頻度 | 影響範囲 | 定義 |
| :--- | :--- | :--- | :--- | :--- |
| **Gross Domestic Product** | GDP | 四半期 (14日) | 全セクター | 国内総生産。経済活動の総量。 |
| **Consumer Price Index** | CPI | 四半期 (14日) | 金利, 債券 | 消費者物価指数。インフレ率の基準。 |
| **Unemployment Rate** | UER | 四半期 (14日) | 消費財セクター | 失業率。労働市場の健全性。 |
| **Interest Rate** | IR | 随時 (中央銀行) | 全資産 | 政策金利。中央銀行による金融調節。 |
| **Consumer Confidence** | CCI | 週次 | 小売, サービス | 消費者信頼感指数。マインドセット。 |

## 2. GDP (Gross Domestic Product)

### 2.1 定義と計算式
支出面からのアプローチを採用し、ゲーム内の以下の要素を集計して算出します。

$$GDP = C + I + G + (X - M)$$

*   **C (Consumption)**: 民間消費
    *   `Sum(B2C_Sales)`: 全企業のB2C売上総額。
    *   `Sum(Service_Fees)`: プレイヤーが支払った各種手数料（取引手数料除く）。
*   **I (Investment)**: 民間投資
    *   `Sum(Corporate_CapEx)`: 企業の設備投資額（生産能力増強）。
    *   `Sum(Inventory_Change)`: 在庫変動額（期末在庫 - 期首在庫）。
*   **G (Government Spending)**: 政府支出
    *   `Sum(B2G_Sales)`: 企業から政府への売上（公共事業、防衛装備品など）。
    *   `Sum(Basic_Income)`: プレイヤーへのベーシックインカム支給総額。
*   **X (Exports)**: 輸出
    *   他国（リージョン外）への販売額。※現状はグローバル市場のため、地域間貿易として計算。
*   **M (Imports)**: 輸入
    *   他国（リージョン外）からの購入額。

### 2.2 変動要因
*   **企業の生産活動**: 企業の稼働率が上がり、売上が増えればGDPは成長します。
*   **プレイヤーの消費**: アイテム購入やサービス利用が活発になればCが増加します。
*   **政府の財政出動**: 不況時の公共事業（Gの増加）はGDPを下支えします。

## 3. CPI (Consumer Price Index) & Inflation

### 3.1 定義と計算式
代表的な「消費者バスケット」の価格変動を指数化します。基準時点の価格を100とします。

$$CPI_t = \frac{\sum (Price_{i,t} \times Weight_i)}{\sum (Price_{i,0} \times Weight_i)} \times 100$$

*   **バスケット構成品目 ($i$)**:
    1.  **Food (30%)**: `Red Ox Food` 等の農産物・加工食品。
    2.  **Energy (20%)**: `Titan Energy` 等の燃料・電力。
    3.  **Goods (30%)**: `Stardust Luxury`, `Silicon Dragon` 等の消費財。
    4.  **Services (20%)**: `Goliath Bank` 等の金融サービス、`Trans-Oceanic` 等の物流費。

### 3.2 インフレ率 (Inflation Rate)
前期比または前年同期比でのCPI上昇率をインフレ率と定義します。

$$Inflation = \frac{CPI_t - CPI_{t-1}}{CPI_{t-1}} \times 100$$

### 3.3 変動メカニズム
*   **コストプッシュ**: 資源価格（エネルギー、穀物）の高騰が、バスケット価格を直接押し上げます。
*   **デマンドプル**: 好景気（GDP成長）により需要が供給を上回ると、企業は値上げを行い、CPIが上昇します。
*   **通貨供給量**: 中央銀行による国債買いオペ（QE）でマネーサプライが増えると、インフレ圧力が生じます。

## 4. Unemployment Rate (失業率)

### 4.1 定義と計算式
労働力人口に対する完全失業者の割合。ゲーム内では「企業の稼働率」と逆相関するモデルを採用します。

$$Unemployment Rate = Natural Rate + \beta \times (1 - Average Utilization Rate)$$

*   **Natural Rate (自然失業率)**: 構造的に存在する失業率（例: 3.0%）。
*   **Average Utilization Rate**: 全企業の平均設備稼働率（0.0〜1.0）。
*   **$\beta$**: 感応度係数（オークンの法則に基づく）。

### 4.2 影響
*   **消費への影響**: 失業率が上昇すると、GDPの `C (Consumption)` が減少します。特に贅沢品（Luxury）の需要が激減します。
*   **政策への影響**: 失業率が高止まりすると、政府は「雇用対策（Gの増加）」や「金融緩和（利下げ）」を行う圧力を受けます。

## 5. Interest Rate (政策金利)

### 5.1 決定プロセス (Taylor Rule)
中央銀行AIは、インフレ率とGDPギャップに基づいて政策金利を決定する「テイラー・ルール」に類似したアルゴリズムで動作します。

$$i = r^* + \pi + 0.5(\pi - \pi^*) + 0.5(y - y^*)$$

*   $i$: 政策金利目標値
*   $r^*$: 実質均衡金利（例: 2.0%）
*   $\pi$: 現在のインフレ率
*   $\pi^*$: 目標インフレ率（例: 2.0%）
*   $y$: 実質GDP成長率
*   $y^*$: 潜在GDP成長率

### 5.2 市場への波及
*   **債券市場**: 金利上昇 → 債券価格下落（逆相関）。
*   **株式市場**: 金利上昇 → 企業の借入コスト増加 → 利益圧縮 → 株価下落。また、割引率上昇によりバリュエーション（PER）が低下します。
*   **為替市場**: 金利上昇 → 通貨の魅力度アップ → 通貨高（Appreciation）。

## 6. 指標間の循環サイクル (Economic Cycle)

1.  **好況期 (Boom)**:
    *   需要増 → GDP成長 → 企業業績アップ → 失業率低下。
    *   需給逼迫 → **CPI上昇 (インフレ)**。
2.  **引き締め (Tightening)**:
    *   インフレ抑制のため、中央銀行が **利上げ** を実施。
    *   借入コスト増 → 企業の設備投資(I)抑制 → 消費(C)減速。
3.  **後退期 (Recession)**:
    *   GDP成長鈍化 → 企業業績悪化 → **失業率上昇**。
    *   需要減 → 物価下落 (ディスインフレ/デフレ)。
4.  **緩和・回復 (Recovery)**:
    *   景気刺激のため、中央銀行が **利下げ** を実施。
    *   資金調達コスト低下 → 投資・消費の再開 → 好況期へ。

## 7. 国家別特性 (Country-Specific Characteristics)

Paper Street の世界設定 (`world_setting.md`) に基づき、各国の経済指標には異なる「クセ」や「感応度」が設定されています。
これにより、同じ世界的イベントに対しても、国ごとに異なる反応を示します。

### 7.1 Arcadia (Tech Utopia)
*   **CPIバスケット**: `Services (40%)`, `Goods (30%)`, `Energy (20%)`, `Food (10%)`
    *   高度なデジタル社会であるため、サービス価格の影響を強く受けます。食料自給率は低いですが、エンゲル係数も低いため食料価格の影響は限定的です。
*   **労働市場**:
    *   **Natural Unemployment Rate**: **4.5%** (高め)
    *   AIによる自動化が進んでいるため、構造的失業率が比較的高く設定されています。
*   **金利感応度**:
    *   テック企業が多いため、金利上昇に対する株価の下落幅（デュレーション）が最も大きくなります。

### 7.2 Boros Federation (Industrial Giant)
*   **CPIバスケット**: `Energy (40%)`, `Food (30%)`, `Goods (20%)`, `Services (10%)`
    *   エネルギー多消費型の産業構造であるため、原油価格（Titan Energy）の変動がCPIに直結します。
*   **GDP**:
    *   **輸出依存度**: 極めて高い。
    *   世界景気（他国のGDP成長率）との相関が高く、世界的な不況時には最もダメージを受けます。
*   **通貨政策**:
    *   **通貨安バイアス**: 輸出競争力維持のため、自国通貨高（BRB高）局面では、中央銀行が積極的に「売り介入」を行います。

### 7.3 El Dorado (Resource Kingdom)
*   **CPIバスケット**: `Goods (50%)`, `Services (20%)`, `Energy (10%)`, `Food (20%)`
    *   国内産業が未成熟で、消費財の多くを輸入（Goods）に頼っているため、輸入インフレ（為替安＝物価高）が起きやすい構造です。
*   **GDP**:
    *   **資源価格連動**: レアメタルやリチウムの国際価格とGDPがほぼ完全に連動します（相関係数 > 0.9）。
*   **ダッチ・ディジーズ (Dutch Disease)**:
    *   資源高で自国通貨（DRL）が急騰すると、国内の他の産業（観光や軽工業）が壊滅的な打撃を受け、失業率が急増するリスクがあります。

### 7.4 Neo Venice (Financial Hub)
*   **GDP**:
    *   **金融セクター依存**: `Goliath Bank` や `Shadow Fund` の業績、および市場の取引量（Volume）がGDPの大部分を構成します。
    *   **資産効果 (Wealth Effect)**: 株価や仮想通貨価格の上昇が、直接的に個人消費(C)を押し上げる係数が高く設定されています。
*   **金利政策**:
    *   **流動性重視**: インフレ抑制よりも、市場のクラッシュを防ぐことを優先し、危機時には迅速に利下げや量的緩和を行います。

### 7.5 San Verde (Agri-Republic)
*   **CPIバスケット**: `Energy (30%)`, `Goods (40%)`, `Services (10%)`, `Food (20%)`
    *   食料生産国ですが、肥料や燃料（Energy）の輸入コストが食料価格に転嫁されるため、エネルギー価格への感応度が高いです。
*   **季節性 (Seasonality)**:
    *   収穫期（Q2, Q4など）にGDPがスパイクする季節変動特性を持ちます。
*   **安定性**:
    *   世界的な不況下でも食料需要は落ちないため、GDPのボラティリティは最も低く、ディフェンシブな投資先となります。

### 7.6 Novaya Zemlya (Energy Frontier)
*   **CPIバスケット**: `Food (40%)`, `Goods (40%)`, `Services (10%)`, `Energy (10%)`
    *   極寒の地であるため、食料と生活必需品（Goods）のほぼ全てを輸入に依存しており、輸入インフレ耐性が極めて低いです。一方でエネルギーコストは安価で安定しています。
*   **GDP**:
    *   **エネルギー輸出とデータセンター**: 電力の輸出（Titan Energy, Atomos Energy）と、安価な電力を求めて集まったデータセンター群（Service Export）がGDPを牽引します。
*   **リスク要因**:
    *   **温暖化リスク**: 環境温度の上昇は、冷却効率の悪化（データセンターのコスト増）や永久凍土融解によるインフラ崩壊を招き、GDPを直接毀損します。

### 7.7 Pearl River Zone (Cyber-Industrial Hub)
*   **CPIバスケット**: `Energy (40%)`, `Food (30%)`, `Services (20%)`, `Goods (10%)`
    *   24時間稼働の工場群を維持するための電力需要が凄まじく、エネルギー価格の高騰がインフレに直結します。
*   **労働市場**:
    *   **フィリップス曲線のスティープ化**: 労働市場が極めて流動的で、好況時（求人増）には賃金が急騰しやすい傾向があります。
*   **GDP**:
    *   **Techサイクル連動**: 世界のテック需要（Arcadiaの株価や製品サイクル）に対して、部品供給拠点として遅行指数的な相関（Lagging correlation）を持ちます。
    *   **パンデミックリスク**: 人口過密により、新型ウイルス発生時には都市封鎖（Lockdown）が行われ、GDPが瞬間的にゼロ近くまで落ち込むテールリスクがあります。
